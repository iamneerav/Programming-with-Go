# Understanding Interfaces and Polymorphism (Part 2)

## Purpose
Understand interfaces and polymorphism in Go through concrete types and shared behavior.

## What I Practiced
- Defining a `Pet` interface.
- Creating `Dog` and `Owner` types with methods.
- Using embedded structs.
- Passing different concrete types into the same function through an interface.

## Run
```bash
go run .
```

## Notes
- Practiced defining interfaces and implementing methods.
- Reinforced polymorphism by using different concrete types through one interface.
- Writing my own example clarified behavior-driven design in Go.
- The program demonstrates both direct method calls and interface-based function calls.
